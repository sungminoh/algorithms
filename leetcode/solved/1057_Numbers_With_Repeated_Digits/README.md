### [1057. Numbers With Repeated Digits](https://leetcode.com/problems/numbers-with-repeated-digits/description/)

Hard

Given an integer `` n ``, return _the number of positive integers in the range _`` [1, n] ``_ that have __at least one__ repeated digit_.

 

<strong class="example">Example 1:</strong>

```
Input: n = 20
Output: 1
Explanation: The only positive number (<= 20) with at least 1 repeated digit is 11.
```

<strong class="example">Example 2:</strong>

```
Input: n = 100
Output: 10
Explanation: The positive numbers (<= 100) with atleast 1 repeated digit are 11, 22, 33, 44, 55, 66, 77, 88, 99, and 100.
```

<strong class="example">Example 3:</strong>

```
Input: n = 1000
Output: 262
```

 

__Constraints:__

*   <code>1 <= n <= 10<sup>9</sup></code>
