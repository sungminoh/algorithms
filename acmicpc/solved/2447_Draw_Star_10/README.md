## 문제

예제를 보고 별찍는 규칙을 유추한 뒤에 별을 찍어 보세요.

## 입력

첫째 줄에 N이 주어진다. N은 항상 3의 제곱꼴인 수이다. (1, 3, 9, 27, ...) (N=3^k, 0<=k<8)

## 출력

첫째 줄부터 N번째 줄까지 별을 출력한다.

## 예제 입력 복사

```
27
```
